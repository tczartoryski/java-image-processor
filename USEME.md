# Command examples:
### (Called from the command line)

* **load** (Filepath to image) (image-name) --> Loads an image into the camera roll
* **save** (Filepath to image) (image-name) --> Saves a specified image 
* **red-component** (image-name) (dest-image-name) --> Applies the red-component to an image 
* **green-component** (image-name) (dest-image-name) --> Applies the green-component to an image 
* **blue-component** (image-name) (dest-image-name) --> Applies the blue-component to an image 
* **value-component** (image-name) (dest-image-name) --> Applies the value-component to an image 
* **intensity-component** (image-name) (dest-image-name) --> Applies the intensity-component to an image 
* **luma-component** (image-name) (dest-image-name) --> Applies the luma-component to an image 
* **horizontal-flip** (image-name) (dest-image-name) --> Flips an image horizontally 
* **vertical-flip** (image-name) (dest-image-name) --> Flips an image vertically
* **brighten** (increment) (image-name) (dest-image-name) --> Brightens an image by a specified increment
* **sharpen** (image-name) (dest-image-name) --> Sharpens an image
* **blur** (image-name) (dest-image-name) --> Blurs an image
* **sepia** (image-name) (dest-image-name) --> Applies a sepia filter to an image
* **greyscale** (image-name) (dest-image-name) --> Applies a greyscale visualization to an image (automatically uses luma)
* **quit** --> Quits the program
